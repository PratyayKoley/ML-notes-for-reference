# Types of Supervised Learning

**Classification** :
It is about predicting a class or discrete values.

- Male or female
- True or False
- Dog or Cat

**Regression** :
It is about predicting a quantity or continuous values.

- Salary
- Age
- Rainfall in cm according to current temperature
